# Afisha_YandexДанный проект посвящен анализу лояльности пользователей платформы Яндекс Афиша
Цель исследования — выявить ключевые факторы, влияющие на удержание и активность аудитории, а также определить сегменты наиболее лояльных пользователей. Данный анализ поможет лучше понять поведение аудитории, повысить эффективность маркетинговых стратегий и улучшить пользовательский опыт на платформе Яндекс Афиша.
